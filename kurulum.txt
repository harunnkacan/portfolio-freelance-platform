1. SİSTEM GEREKSİNİMLERİ:
--------------------------------------------------------------------------------
- Bir Cloudflare hesabı (Workers & KV kullanımı için)
- Bun (v1.x) veya Node.js (v18+) kurulu terminal
- OpenRouter API Anahtarı (AI Özellikleri için opsiyonel)
2. YAYINA ALMA (DEPLOYMENT):
--------------------------------------------------------------------------------
Scripti Cloudflare Workers üzerinde yayına almak için terminale şunları yazın:
$ bun install          # Bağımlılıkları yükle
$ bun run deploy       # wrangler login sonrası otomatik yayına alım
3. ADMİN GİRİŞ BİLGİLERİ:
--------------------------------------------------------------------------------
Platform yönetim paneline erişmek için:
URL: /auth
E-Posta: admin@lumina.blog
Şifre: 123456 (Mock)
Admin paneline girdikten sonra sağ üstteki "KULLANICI MODUNA DÖN" linkini
takip ederek siteyi admin yetkileriyle (Düzenleme butonları aktif) gezebilirsiniz.
4. YAPILANDIRMA (CONFIG):
--------------------------------------------------------------------------------
- Site ayarları Zustand persist middleware ile tarayıcıda saklanır.
- Cloudflare R2 dosya yöneticisi arayüzü simüle edilmiştir.
- OpenRouter API anahtarınızı Admin > AI Asistan sekmesinden ekleyebilirsiniz.
5. DOSYA YAPISI:
--------------------------------------------------------------------------------
/src/lib/content.ts      -> Ana veri kaynağı (CMS)

/worker/index.ts         -> Cloudflare Workers backend mantığı
