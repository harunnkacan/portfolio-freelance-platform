1. GENEL BAKIŞ:
--------------------------------------------------------------------------------
Lumina, modern web teknolojileriyle geliştirilmiş, "Cyber Red" estetiğine sahip
bir portfolyo, blog ve dijital market çözümüdür. Yazılım geliştiriciler ve
dijital içerik üreticileri için optimize edilmiştir.
2. TEKNİK ÖZELLİKLER:
--------------------------------------------------------------------------------
- Framework: React (Vite) + TypeScript
- Styling: Tailwind CSS (Custom Cyber Red Palette)
- State: Zustand (Persistent)
- Animasyon: Framer Motion
- Admin Paneli: Kapsamlı (İçerik, Market, Kullanıcı, AI, Medya, Loglar)
- AI Entegrasyonu: OpenRouter (GPT-4o, Claude 3.5 desteği)
3. KURULUM VE YAYIN (DEPLOYMENT):
--------------------------------------------------------------------------------
Lumina, Cloudflare Workers & Assets üzerinde 1 dakikada yayına alınabilir:
$ bun install          # Bağımlılıkları yükle
$ bun run deploy       # wrangler ile yayına al (Wrangler login gerektirir)
4. YÖNETİCİ VE ÜYE YÖNETİMİ:
--------------------------------------------------------------------------------
- URL: /auth
- Varsayılan Admin: admin@lumina.blog
- Varsayılan Şifre: 123456 (Simüle edilmiş)
- Audit Logs: Sistemdeki her kritik eylem (giriş, güncelleme, silme) 
  Admin > Sistem Günlüğü sekmesinde tarih, kullanıcı ve detayla kaydedilir.
5. SİSTEM YEDEĞİ VE RESTORE:
--------------------------------------------------------------------------------
- Site Ayarları Yedeği: Admin > Site Ayarları > SİSTEM YEDEĞİ (JSON)
- Audit Log Çıktısı: Admin > Sistem Günlüğü > DIŞA AKTAR (JSON)
- Bu dosyalar yerel tarayıcı hafızasını sıfırladığınızda ayarları geri 
  yüklemek için kullanılabilir.
6. AI ASİSTAN YAPILANDIRMASI:
--------------------------------------------------------------------------------
- OpenRouter API anahtarınızı Admin > AI Asistan sekmesine girin.
- Model ve Temperature ayarlarını buradan yönetebilirsiniz.
- Makale detay sayfalarında "AI Assistant" bloğu otomatik aktifleşecektir.
